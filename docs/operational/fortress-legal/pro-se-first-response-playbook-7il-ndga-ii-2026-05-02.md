# Pro Se First-Response Playbook - 7IL Case II

Date: 2026-05-02
Matter: 7IL Properties, LLC v. Knight / James, N.D. Ga. Case II
Operator service status: not served as of 2026-05-02

This playbook is for readiness. It is not legal advice and should not be filed as-is. If counsel is retained, counsel should review all response strategy before filing.

## 1. Trigger Events

Treat any of these as a P0 event:

- Personal service of summons and complaint
- Service at dwelling or usual place of abode
- Delivery to an authorized agent
- Waiver of service request
- Docket entry suggesting service or proof of service
- Email/letter from plaintiff counsel claiming service
- Court notice about service deadline or default risk

## 2. Immediate Actions If Served

Same day:

- Record exact date, time, place, method, and person served.
- Scan or photograph all papers received.
- Save envelope, tracking, affidavits, and process-server details.
- Update the matter command center.
- Calculate response deadline conservatively.
- Do not communicate substantive strategy to plaintiff counsel without deciding counsel/pro-se posture.

Within 24 hours:

- Compare served complaint to the curated complaint in the package.
- Check whether summons is signed, sealed, names parties correctly, and identifies response timing.
- Check docket for proof of service.
- Notify any conflict-cleared counsel candidates that service occurred and deadline is active.

## 3. Response Options To Preserve

Potential response paths to evaluate before filing:

- Answer with affirmative defenses.
- Rule 12 motion before answer, if appropriate.
- Answer plus counterclaims.
- Motion plus preserved counterclaim strategy.
- Extension request if counsel is being retained or package review is underway.

Do not choose the response path casually. Some defenses can be waived if not raised at the right time.

## 4. Answer Matrix Template

| 1 |  | Admit / Deny / Lack knowledge |  |  |  | Yes |  |

Response options should be controlled:

- Admit only if fully true and harmless.
- Deny if materially false or misleading.
- Lack knowledge only if genuinely unable to admit or deny after reasonable review.
- Qualified admission if part is true but framing is misleading.

## 5. Affirmative Defense Worklist

Evaluate, do not blindly assert:

- Claim preclusion / res judicata
- Collateral estoppel
- Failure to state a claim
- Lack of enforceable easement / facial unenforceability
- Waiver
- Estoppel
- Laches
- Unclean hands
- Failure to mitigate
- Statute of limitations, if applicable
- Privilege / immunity defenses, if applicable
- Reservation of defenses pending discovery

Each defense needs an evidence or legal basis before filing.

## 6. Counterclaim Worklist

Evaluate and evidence-map:

- Abuse of process against 7IL Properties, LLC
- Individual-capacity theory against John Thatcher
- Punitive damages theory
- Terry Wilson-related professional negligence / fraud / conspiracy theory, if counsel supports it
- Any declaratory relief tied to easement enforceability
- Fees/costs theory if available

Do not overplead weak claims. A smaller, well-supported counterclaim can be more powerful than a sprawling one.

## 7. Evidence Traceability Rule

Every factual assertion in a draft answer, motion, or counterclaim should map to at least one source:

- Complaint paragraph
- Docket item
- Case I filing/order
- Easement instrument
- Email/document evidence
- Witness or operator fact record
- NAS source path

If the assertion cannot be traced, it is not filing-ready.

## 8. Red-Team Questions Before Filing

- What would plaintiff say is waived if this filing goes out?
- What would Judge Story find overstated or unsupported?
- Which allegations should be denied more narrowly?
- Which defenses need stronger factual grounding?
- Does any paragraph disclose privileged thinking unnecessarily?
- Are counterclaims supported by facts, not just anger at the litigation?
- Does the filing preserve leverage without overcommitting?

## 9. Counsel Review Ask

If counsel clears conflicts, the most efficient first ask is:

Please review the attached package and first-response matrix for (1) answer posture, (2) Rule 12 posture, (3) affirmative defenses, (4) counterclaim viability, and (5) waiver risk. I am looking for a Phase 1 scoped review before first responsive filing, not an open-ended engagement unless we mutually agree the fit and economics support it.
