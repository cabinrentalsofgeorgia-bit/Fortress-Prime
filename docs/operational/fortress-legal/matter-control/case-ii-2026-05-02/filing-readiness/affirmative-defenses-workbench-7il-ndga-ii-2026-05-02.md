# Affirmative Defenses Workbench - 7IL Case II

Date: 2026-05-02
Operator service status: not served as of 2026-05-02

This is an evaluation list, not a filing. Affirmative defenses should be asserted only if there is a good-faith factual and legal basis after reviewing the complaint, exhibits, Case I record, contracts, closing documents, title materials, and counsel input where available.

## Pre-Filing Defense Test

For each defense, answer:

1. What paragraph/count does it address?
2. What fact supports it?
3. What document supports that fact?
4. Is it waived if omitted from the first response?
5. Would counsel assert it as written?
6. Would Judge Story view it as grounded and useful?
