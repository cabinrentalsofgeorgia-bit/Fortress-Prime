## Codes

| Code | Meaning |
|---|---|
| C1 | Amendment #2 says the item was fixed, but independent completion proof is not yet pinned. |
| C2 | Amendment #2 says the item was fixed and gives method detail; preserve the method but still pin independent proof. |
| C3 | Amendment #2 says nothing needed; do not treat as a repair item unless plaintiff identifies another written duty. |
| C4 | Source description is not text-extracted yet; manual page/image review controls before merits use. |
| S1 | General repair survival gate: PSA source text points to pre-closing repair performance; no item-specific post-closing survival/reservation is pinned yet. |
| S2 | Maintenance or consumable gate: same-condition and post-closing recurrence/maintenance proof matters before exposure can be assessed. |
| S3 | Safety/system gate: exact location, independent completion proof, survival/reservation, and same-condition proof are all required. |
| S4 | No-obligation gate: first issue is whether the item was ever a written obligation. |

## Control Rule

The working chain is: 2021 inspection item -> Amendment #2 written filter -> independent completion proof -> judgment/closing survival or reservation -> same-condition proof -> recoverable damages analysis.

Do not accept an item as live simply because it appeared in an inspection report or a comparison spreadsheet.
