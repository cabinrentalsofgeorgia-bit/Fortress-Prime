# Fortress Legal Agent Operating Context

This directory contains deterministic, machine-readable startup context for AI agents and Codex sessions.

The context is operational guidance only:

- It does not authorize counsel signoff.
- It does not create final legal conclusions.
- It does not authorize filing, service, sending, email, or external submission.
- It does not override unresolved-source exclusions.
- It must not contain secrets, auth state, confidential legal text, or restricted-content body text.

Regenerate with:

```bash
node fortress-guest-platform/scripts/operational-memory/generate-agent-context.mjs
```
